# ScreenEdit

Style your screenshots in seconds. Upload an image, add a background (solid/gradient/transparent), rounded corners, shadow — then download PNG or copy to clipboard.

**→ https://screenedit.online**

## Features

- Drag & drop, click upload, or **Ctrl+V** paste
- Background: Solid color, Gradient (with angle control), or Transparent
- Canvas size, padding, border radius (bg + image), shadow
- EN / RU interface
- Dark theme
- All processing in browser — nothing leaves your machine

## Run locally

```bash
npm install
npm run dev
```
